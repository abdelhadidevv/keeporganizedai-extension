# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - April 21, 2026

### Added
- Initial release of KeepOrganizedAI
- AI-powered bookmark categorization using Gemini, Claude, or OpenAI
- 3-step wizard workflow for organizing bookmarks
- Folder locking system:
  - Hard Lock: Protect folders entirely from changes
  - Smart Lock: Preserve folders as category containers
- Automatic backup before any changes
- Global search functionality
- Dark/Light theme support (follows system preference)
- Chrome Extension Manifest V3 implementation
- Comprehensive development tooling:
  - Vite for fast development and building
  - TypeScript with strict mode
  - ESLint (Airbnb config) + Prettier for code quality
  - Vitest + React Testing Library for testing
  - Husky for pre-commit hooks
- Contributing guidelines and issue templates

### Tech Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- Radix UI (accessible components)
- dnd-kit (drag and drop)
